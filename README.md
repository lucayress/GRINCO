# GRINCO
Original implementation of **GRINCO**, a group-invariant coreset framework for data-efficient active learning in quotient spaces.

Code for the paper "*Group-invariant Coresets for Data-efficient Active Learning*" by Luciano C. Ayres, José C. M. Bermudez, Sérgio J. M. de Almeida, and Ricardo A. Borsoi.

## Status

The manuscript is currently under review.

To avoid changing the public record during submission and peer review, the source code is **not public yet**. This repository will be populated after the review process is finished.
